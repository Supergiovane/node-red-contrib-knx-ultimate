L’onglet **KNX Debug** dans la barre latérale affiche une vue en temps réel du journal interne de KNX Ultimate.

<img src="https://raw.githubusercontent.com/Supergiovane/node-red-contrib-knx-ultimate/master/img/wiki/knx-debug.png" alt="KNX Debug sidebar" style="width:100%;max-width:980px;border-radius:14px;box-shadow:0 12px 26px rgba(0,0,0,0.18);" />

## À quoi ça sert

- Comprendre rapidement ce que fait KNX Ultimate (erreurs, avertissements, info, debug).
- Auto-refresh + copier dans le presse‑papiers.

## Activation

Fourni par le plugin Node-RED :

- `package.json` → `node-red.plugins.knxUltimateMonitorSidebar`
- Fichier : `nodes/plugins/knxUltimateMonitor-sidebar-plugin.html`
