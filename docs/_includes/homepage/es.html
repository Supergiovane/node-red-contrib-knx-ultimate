Node-RED se encuentra con KNX
Crea automatizaciones KNX profesionales más rápido: la combinación perfecta de simplicidad y potencia.

KNX-Ultimate une la flexibilidad de Node-RED con una integración KNX/IP de nivel empresarial: tunneling seguro, routing, Hue v2, sincronización de contexto, watchdogs, diagnóstico y mucho más, listo para proyectos reales.

Instalar desde npm Ver en GitHub

Tecnologías compatibles

  • Núcleo KNX
    KNX Tunnelling

    Tunnelling estilo ETS con reconexión automática, múltiples sesiones seguras y transmisión de telegramas de alto rendimiento.

    ✅ Fully supported
  • Espina dorsal del bus
    KNX Routing

    Routing multicast a través de routers KNX/IP con limitación y protección frente a tormentas de telegramas en instalaciones grandes.

    ✅ Fully supported
  • Transporte seguro
    KNX IP Secure / Data Secure

    Gestión completa de handshakes, certificados y claves de sesión para tunnelling y cargas protegidas.

    ✅ Fully supported
  • Dispositivos TP
    Serial FT1.2 (TPUART / KBerry)

    Controla TPUART HAT, módulos Weinzierl KBerry/BAOS y 307x, y otras interfaces TP/FT1.2 mediante /dev/tty*. El selector Gateway lista los puertos serie y aplica los parámetros UART recomendados al elegir uno. Soporta KNX TP Data Secure (mediante keyring ETS) y es totalmente compatible con instalaciones que también usan KNX IP Secure / Data Secure en el lado IP.

    ✅ Fully supported
  • Hue e IoT
    Philips Hue v2 + IoT

    Escenas, sensores y luminarias Hue v2 integrados con pasarelas IoT hacia servicios como MQTT, InfluxDB y más.

    ✅ Fully supported

Importar CSV desde ETS

Importa nombres de canales, direcciones de grupo y etiquetas desde exportaciones ETS directamente en el nodo KNX-Ultimate.

  • Compatibilidad con CSV/ESF ETS y knxproj
  • Alineación de contexto y tipado automático
  • Validación de GAs duplicadas o faltantes

Empieza con Herramientas → Importar ETS dentro del nodo.
