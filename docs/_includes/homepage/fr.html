Node-RED rencontre KNX
Créez plus vite des automatisations KNX professionnelles : le parfait équilibre entre simplicité et puissance.

KNX-Ultimate allie la flexibilité de Node-RED à une intégration KNX/IP de niveau entreprise : tunneling sécurisé, routage, Hue v2, synchronisation du contexte, watchdogs, diagnostics et plus encore—prêt pour vos projets réels.

Installer depuis npm Voir sur GitHub

Technologies prises en charge

  • Cœur KNX
    KNX Tunnelling

    Tunnelling façon ETS avec reconnexion automatique, sessions multiples sécurisées et diffusion de télégrammes haute performance.

    ✅ Fully supported
  • Infrastructure bus
    KNX Routing

    Routage multicast via routeurs KNX/IP avec limitation de débit et protection contre les tempêtes de télégrammes.

    ✅ Fully supported
  • Transport sécurisé
    KNX IP Secure / Data Secure

    Gestion complète des handshakes, certificats et clés de session pour tunnelling et charges sécurisées.

    ✅ Fully supported
  • Périphériques TP
    Serial FT1.2 (TPUART / KBerry)

    Pilotez TPUART HAT, modules Weinzierl KBerry/BAOS et 307x, ainsi que d'autres interfaces TP/FT1.2 via /dev/tty*. Le sélecteur Gateway liste les ports série et applique les paramètres UART recommandés. Prend en charge KNX TP Data Secure (via keyring ETS) et reste entièrement compatible avec des installations qui utilisent aussi KNX IP Secure / Data Secure côté IP.

    ✅ Fully supported
  • Hue & IoT
    Philips Hue v2 + IoT

    Intégration native des scènes, capteurs et luminaires Hue v2 ainsi que des passerelles vers MQTT, InfluxDB et d’autres services.

    ✅ Fully supported

Import CSV depuis ETS

Importez noms de canaux, adresses de groupe et tags issus des exports ETS directement dans le nœud KNX-Ultimate.

  • Prise en charge des CSV/ESF ETS et knxproj
  • Alignement du contexte et typage automatique
  • Validation des GA en doublon ou manquantes

Ouvrez Outils → Import ETS directement dans le nœud.

Bonnes pratiques

Séparez logique et visualisation dans des flows indépendants, synchronisez les données essentielles via le contexte global et versionnez vos exports ETS.
